Dans un jeu de mots croisés, le joueur remplit un rectangle avec des carrés noirs et blancs, où chaque ligne et chaque colonne vides doivent être remplies par un mot. Les mots horizontaux se croisent avec les mots verticaux en lettres communes. Dans cette mission, seuls deux mots sont donnés.

Étant donné un mot horizontal et un mot vertical, votre tâche consiste à trouver la meilleure lettre de croisement entre eux, définie comme la lettre qui permet le croisement le plus à droite dans le mot horizontal et, en cas d'égalité, le croisement le plus bas dans le mot vertical.

Vous devez renvoyer un tuple de deux entiers: les indices basés sur 1 de la lettre commune dans les mots horizontaux et verticaux respectivement. S'il n'y a pas de lettre commune, renvoyez -1, -1.

Entrée: Deux chaînes de caractères en majuscules (str{% if is_js %}ing{% endif %}).

Sortie: Tuple de deux entiers ({% if is_js %}number{% else %}int{% endif %}).
